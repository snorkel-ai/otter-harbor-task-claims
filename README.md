# Otter Harbor Claims

A lightweight dashboard for browsing Harbor tasks, claiming one task at a time, submitting completed work, and tracking review status.

## How to use the dashboard

### 1. Open the dashboard

Go to the Harbor Task Claims dashboard.

You will see a grid of task cards. Each card shows the task status, domain, paper title, task ID, and a short preview.

### 2. Sign in with GitHub

Click **Sign in with GitHub**.

You need to be signed in before you can claim or submit a task.

### 3. Browse available tasks

Use the search box or status filter to find a task.

Task statuses include:

* **Available**: open for claiming
* **Claimed**: currently claimed by someone
* **Submitted**: completed and waiting for admin review
* **Approved**: reviewed and accepted
* **Blocked**: needs admin attention

### 4. Open a task

Click **View details** on a task card.

This opens the full task view, including:

* Task concept
* Suggested Harbor steps
* Verifier ideas
* Source links
* Current claim status

### 5. Claim a task

Click **Claim task**.

You may only have **one active claim at a time**.

Claims expire after **24 hours** if they are not submitted.

### 6. Complete the task

After claiming a task, complete the work.

### 7. Submit your work

Return to the dashboard.

In the **Your active claim** panel:

1. Paste your **Snorkel Submission ID**.
2. Click **Submit**.

Your task will move to **Submitted** while it waits for admin review.

### 8. Release a task if needed

If you no longer want to work on a claimed task, click **Release**.

This returns the task to the available pool so someone else can claim it.
